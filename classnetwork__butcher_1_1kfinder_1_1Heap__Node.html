Detailed Description

template<typename T, typename Comparison, std::size_t Max_Children = 2>
class network_butcher::kfinder::Heap_Node< T, Comparison, Max_Children >

Simple node class, used to construct a min-Heap (though pointers...). Thus, copies are allowed, but not moves. Depth can be used to keep track of the number of nodes the current node in each branch.

Template Parameters
TThe node content
ComparisonContent comparison struct
Max_ChildrenThe number of children per node

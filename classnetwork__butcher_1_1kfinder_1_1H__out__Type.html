Detailed Description

template<typename T, typename Comparison>
class network_butcher::kfinder::H_out_Type< T, Comparison >

Simple class used to represent an H_out, a 2-heap with the extra restriction that the first node may have up to a single child

Template Parameters
TThe type of the content of the nodes
ComparisonThe comparison function to use to compare the content of the nodes

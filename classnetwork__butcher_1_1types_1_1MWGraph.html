Detailed Description

template<bool Parallel_Edges, typename Template_Node_Type = Node, typename t_Weight_Type = Time_Type>
class network_butcher::types::MWGraph< Parallel_Edges, Template_Node_Type, t_Weight_Type >

A custom graph class. It contains a single graph and multiple weight collections. Technically, it can be viewed as a collection of graphs with the same structure, but different weight collections.

Template Parameters
Parallel_EdgesIf true, the graph will allow parallel edges
Template_Node_TypeThe type of the node
t_Weight_TypeThe type of the weight

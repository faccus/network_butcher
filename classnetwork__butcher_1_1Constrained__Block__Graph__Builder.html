Detailed Description

template<typename GraphType>
class network_butcher::Constrained_Block_Graph_Builder< GraphType >

This class will be used to generate a block graph from a graph, applying the provided constraints. It will take the input graph and the parameters as constant reference and, based on the graph structure, it will generate the block graph. The constraints are applied after the block graph is generated. Weight importers can be used to import weights from external sources directly in the block graph.
